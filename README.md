# Secure Website

Built using Flask, see https://flask.palletsprojects.com/en/2.2.x/

## Instructions
     
  1. Install Requirements
  
     ```
	 pip install -r REQUIREMENTS.txt
	 ```
	
  2. Run
  
     ```
	 flask --app app/ --debug run
	 ```

The site should now be visible on 127.0.0.1:5000
